# 1. Claude.md 읽고 Claude Custom Command 생성해줘.

## 1-1.claude/commands 디렉토리에 Claude Custom Command가 md 파일로 생성될 때 기본 명령어와 동일하게 생성되는 경우가 있다. 예) status.md, help.md, info.md 등. 이런 파일들은 기본 명령어와 충돌하므로 생성하지 마.

## 1-2 이미 만들어진 Custom Command들은  commands 하위에 디렉토리를 만들어서 그 안에 md 파일을 옮겨주고, md 파일의 이름 앞에 디렉토리 명을 접두사로 붙여줘. 예를 들어 commands/tdd/tdd_go.md 이런 식으로.
