# PPT Coach - 요구사항 정의서

## 1. 프로젝트 개요

### 1.1 프로젝트명
**PPT Coach** - PowerPoint 실시간 가이드 & AI 피드백 시스템

### 1.2 목적
PowerPoint 초보자가 결과완료보고서를 작성할 때, 화면 위에서 단계별로 안내하고 실시간 피드백을 제공하여 고품질 문서를 완성할 수 있도록 지원

### 1.3 대상 사용자
- PowerPoint 작성 경험이 적은 직장인
- 보고서 양식을 처음 접하는 신입사원
- 체계적인 PPT 작성법을 배우고 싶은 사람

## 4. 유스케이스

### UC-05: AI 피드백 요청

**액터**: 사용자  
**사전조건**: 슬라이드 작성 중, 인터넷 연결됨  
**기본 흐름**:
1. 사용자가 "AI 피드백" 버튼을 클릭한다
2. 앱이 현재 슬라이드의 텍스트/구조 정보를 수집한다
3. 앱이 슬라이드 스크린샷을 캡처한다 (선택적)
4. 앱이 AI API에 분석 요청을 전송한다
5. 로딩 인디케이터를 표시한다
6. AI 응답을 수신하여 패널에 표시한다
7. 피드백 항목별로 "적용" 버튼을 제공한다

**대안 흐름**:
- 4a. API 호출 실패 시
  - 4a1. 오류 메시지 표시
  - 4a2. 재시도 버튼 제공
- 6a. 피드백이 너무 긴 경우
  - 6a1. 요약 먼저 표시
  - 6a2. "자세히 보기" 확장 옵션

**결과**: 사용자가 AI의 전문적인 조언을 받아 슬라이드 품질 향상

### UC-06: 특정 단계로 이동

**액터**: 사용자  
**사전조건**: 가이드 진행 중  
**기본 흐름**:
1. 사용자가 패널의 진행 표시줄 또는 목차를 클릭한다
2. 앱이 전체 단계 목록을 팝업으로 표시한다
3. 각 단계의 완료/미완료 상태가 표시된다
4. 사용자가 원하는 단계를 선택한다
5. 앱이 해당 단계로 이동하고 관련 슬라이드로 포커스를 옮긴다
6. 오버레이가 새 단계에 맞게 업데이트된다

**대안 흐름**:
- 4a. 이전에 건너뛴 단계를 선택한 경우
  - 4a1. 해당 단계부터 다시 안내 시작

**결과**: 사용자가 원하는 단계를 자유롭게 탐색 가능

---

### UC-07: 완성 문서 최종 검토

**액터**: 사용자  
**사전조건**: 모든 단계 완료  
**기본 흐름**:
1. 마지막 단계 완료 시 "최종 검토" 화면이 표시된다
2. 앱이 전체 슬라이드를 대상으로 종합 검증을 수행한다
3. 검증 결과를 카테고리별로 표시한다 (폰트, 색상, 레이아웃, 내용)
4. 각 항목에 통과(✓) 또는 경고(!) 표시
5. 경고 항목 클릭 시 해당 슬라이드로 이동 및 수정 안내
6. 모든 항목 통과 시 "완료" 축하 메시지 표시
7. "저장" 및 "PDF 내보내기" 버튼 제공

**대안 흐름**:
- 4a. 심각한 문제가 있는 경우
  - 4a1. 빨간색 경고로 강조
  - 4a2. 완료 전 수정 권장 메시지

**결과**: 사용자가 고품질의 완성된 PPT를 얻음

### UC-08: 앱 설정 변경

**액터**: 사용자  
**사전조건**: 앱 실행 중  
**기본 흐름**:
1. 사용자가 설정 아이콘을 클릭한다
2. 설정 다이얼로그가 표시된다
3. 사용자가 원하는 항목을 변경한다:
   - AI API 키 설정
   - 오버레이 투명도 조절
   - 자동 진행 ON/OFF
   - 알림 빈도 설정
   - 테마 (라이트/다크)
4. 사용자가 "저장" 버튼을 클릭한다
5. 설정이 즉시 적용된다

**결과**: 사용자 맞춤 환경 설정 완료
