## 1. 화면 흐름도

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   시작 화면   │ ──→ │ 템플릿 선택  │ ──→ │  가이드 진행  │
└─────────────┘     └─────────────┘     └──────┬──────┘
                                               │
                    ┌──────────────────────────┼──────────────────────────┐
                    │                          │                          │
                    ▼                          ▼                          ▼
            ┌──────────────┐          ┌──────────────┐          ┌──────────────┐
            │  단계별 작업   │ ←──────→ │  AI 피드백   │          │   최종 검토   │
            └──────────────┘          └──────────────┘          └──────────────┘
                    │                                                     │
                    │                                                     ▼
                    │                                            ┌──────────────┐
                    └───────────────────────────────────────────→│     완료     │
                                                                 └──────────────┘
```

## 3. 화면 상세

### 3.1 시작 화면 (SCR-01)

```
┌────────────────────────────────────┐
│                                    │
│           [PPT Coach 로고]          │
│                                    │
│      PowerPoint 실시간 가이드        │
│                                    │
│     ┌────────────────────────┐     │
│     │      새 가이드 시작       │     │
│     └────────────────────────┘     │
│                                    │
│     ┌────────────────────────┐     │
│     │      이어서 진행하기      │     │
│     └────────────────────────┘     │
│                                    │
│  ─────────────────────────────────  │
│  최근 작업                          │
│  • 결과완료보고서 (3/8 단계) - 어제   │

│                                    │
└────────────────────────────────────┘
```

**구성 요소:**
| 요소 | 타입 | 동작 |
|------|------|------|
| 새 가이드 시작 | Button | 템플릿 선택 화면으로 이동 |
| 이어서 진행하기 | Button | 마지막 세션 복구 (없으면 비활성화) |
| 최근 작업 목록 | ListView | 클릭 시 해당 세션 복구 |

### 3.4 오버레이 (SCR-04)

PowerPoint 창 위에 표시되는 투명 레이어

```
┌─────────────────────────────────────────────────────────┐
│ PowerPoint 창                                           │
│  ┌─────────────────────────────────────────────────┐   │
│  │ 리본 메뉴                                        │   │
│  │    ┌─────────────────┐                          │   │
│  │    │ ╔═══════════════╗ │ ← 하이라이트 영역       │   │
│  │    │ ║  디자인 탭    ║ │   (빨간 테두리)         │   │
│  │    │ ╚═══════════════╝ │                        │   │
│  │    └────────┬────────┘                          │   │
│  │             │                                    │   │
│  │      ┌──────▼──────┐                            │   │
│  │      │ 💬 여기를    │  ← 말풍선 툴팁             │   │
│  │      │ 클릭하세요   │                            │   │
│  │      └─────────────┘                            │   │
│  │                                                  │   │
│  │  ┌───────────────────────────────────────────┐  │   │
│  │  │                                           │  │   │
│  │  │              슬라이드 영역                  │  │   │
│  │  │                                           │  │   │
│  │  └───────────────────────────────────────────┘  │   │
│  └─────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────┘
```

**하이라이트 유형:**
| 유형 | 스타일 | 용도 |
|------|--------|------|
| 강조 테두리 | 빨간색 2px, 펄스 애니메이션 | 클릭할 영역 표시 |
| 반투명 마스크 | 검정 50% 투명도 | 주변 영역 흐리게 |
| 화살표 | 애니메이션 포인터 | 방향 안내 |
| 말풍선 | 둥근 사각형 + 꼬리 | 설명 텍스트 |

**오버레이 속성:**
- `WindowStyle`: None
- `AllowsTransparency`: True
- `Background`: Transparent
- `Topmost`: True
- `IsHitTestVisible`: 하이라이트 영역만 True

### 3.5 피드백 알림 (SCR-05)

화면 우측 하단에 토스트 형태로 표시

```
┌─────────────────────────────┐
│ ⚠️ 폰트 불일치 감지          │
│                             │
│ 슬라이드 3에서 '굴림'        │
│ 폰트가 사용되었습니다.        │
│ '맑은 고딕'으로 통일하세요.   │
│                             │
│ [자동 수정]  [무시]  [닫기]  │
└─────────────────────────────┘
```

**알림 유형:**
| 유형 | 아이콘 | 색상 | 자동 닫힘 |
|------|--------|------|----------|
| 성공 | ✅ | 녹색 | 3초 |
| 경고 | ⚠️ | 주황색 | 수동 |
| 오류 | ❌ | 빨간색 | 수동 |
| 정보 | ℹ️ | 파란색 | 5초 |
